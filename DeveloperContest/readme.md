## AMD Pervasive AI Developer Contest 2023

Source: [[hackster.io]](https://www.hackster.io/contests/amd2023) [[pynq.io]](https://discuss.pynq.io/t/participation-in-amd-pervasive-ai-developer-contest/6556)

* There are 3 categories to submit projects to:
    * Generative AI: Utilize AMD ROCm software on hardware like the AMD Instinct MI210 accelerator or Radeon PRO W7900.
    * Robotics AI: Create applications with the Kria KR260 Robotics Starter Kit, using either Vitis AI or PYNQ.
    * PC AI: Develop applications on AMD Ryzen AI-powered PCs.

* Relevance for PYNQ Users:
    * Particularly of interest to our community is the Robotics AI category, which involves the AMD KRIA™ KR260 Robotics Starter Kit. This category relies on both the Kria-PYNQ and DPU-PYNQ open source efforts.
